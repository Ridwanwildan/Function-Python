## Latihan      

Latihan mengubah function biasa menjadi `lambda function`. Ini merupaka function sebelum diubah ke lambda function :            
```bash
import math
a = lambda x: x**2
b = lambda x, y: math.sqrt(x**2 + y**2)
c = lambda *args: sum(args)/len(args)
d = lambda s: "".join(set(s))
```
